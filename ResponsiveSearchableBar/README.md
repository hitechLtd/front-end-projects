# Responsive Searchable Menu

A mobile-first, fully responsive searchable menu that works beautifully on all screen sizes.

## Features

1. Searchable menu with real-time filtering
2. JavaScript-powered search algorithm that filters results based on the user's query
3. Clean, well-styled menu with smooth UX

## Technologies

- HTML5
- CSS3
- Plain Vanilla JavaScript (no frameworks)

## How to Run

1. Open `index.html` in your browser
