# 🎉 Greeting Card Project

A simple and responsive **Birthday Greeting Card** built using **HTML** and **CSS** as part of the **freeCodeCamp Responsive Web Design Certification**.

## 📖 Project Overview

This project displays a modern birthday greeting card with a clean user interface. It features a birthday message and interactive buttons that simulate sending the card or sharing it on social media.

The goal of this project was to practice creating responsive layouts, styling components, and improving UI design using CSS.

## 🚀 Features

- 🎉 Birthday greeting message
- 📱 Responsive card layout
- 🎨 Modern color scheme
- 🟦 Rounded buttons with hover effects
- 💳 Card shadow for depth
- 📐 Centered content using Flexbox

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox

## 📚 What I Learned

While building this project, I learned how to:

- Structure a webpage using semantic HTML.
- Center elements using CSS Flexbox.
- Style buttons with hover effects.
- Apply border radius and box shadows to create modern UI components.
- Use spacing, typography, and alignment to improve readability.
- Create responsive layouts that look good on different screen sizes.
- Organize CSS for cleaner and more maintainable code.

## 🎯 Future Improvements

- Add JavaScript functionality to the buttons.
- Include animations when the card loads.
- Allow users to customize the greeting message.
- Add more greeting card themes (Christmas, Valentine's Day, Graduation, etc.).
- Improve accessibility with better keyboard navigation and ARIA labels.

## 📷 Preview

A clean birthday greeting card featuring:
- A birthday title with celebration emojis.
- A personalized greeting message.
- "Send Card" and "Share on Social Media" buttons.
